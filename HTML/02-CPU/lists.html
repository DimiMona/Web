Списки

Список определений

<dl> ― Defenitions List;
<dt> ― Description Term;
<dd> ― Description Details;
Нумерованный список
― предназначен для перечисления элеметов в точном порядке
Маркированный список
― предназначен для перечисления элеметов в произвольном порядке
Список определений
― предназначен для перечисления терминоа, и разъяснения этих терминов в виде определений;

Списки поддерживают вложенность, НО список Нельзя вкладывать непосредственно в другой список, а можно вкладывать лишь в элемент другого списка

Нумерованный список

<ol> ― Order List
<li> ― List Item (Элемент списка)
  • Элемент 1;
  • Элемент 2;
  • Элемент 3;
  • Элемент 4;
  • Элемент 5;

    • Нумерованный список можно прервать, и продолжить позже.

    1. Элемент 6;
    2. Элемент 7;
    3. Элемент 8;
    4. Элемент 9;
    5. Элемент 10;
